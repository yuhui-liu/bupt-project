◆ listen()

auto Parser::listen ( ) -> ASTNodePtr
private

解析 "listen" 语句。

该函数解析一个 "listen" 语句，该语句包含一个时间限制和一个或多个子句。

返回
ASTNodePtr 返回解析后的 AST 节点。
异常
std::runtime_error如果解析过程中遇到语法错误，将抛出运行时错误。

解析步骤：

  1. 消费 "timelimit" 关键字。
  2. 消费一个数字，作为时间限制。
  3. 消费左大括号 '{'。
  4. 循环解析子句，直到遇到右大括号 '}'。
  5. 消费右大括号 '}'。

该函数创建一个新的 Say 节点，并将解析的内容添加到该节点中。 首先，它会解析第一个标识符或字符串并添加到 Say 节点中。 然后，它会检查是否存在逗号分隔的其他内容，如果存在，则继续解析并添加到 Say 节点中。

◆ statement()

auto Parser::statement ( ) -> ASTNodePtr
private

解析并返回一个语句节点。

该函数根据当前的标记类型解析不同类型的语句节点。

  • 如果标记类型为 SAY，则解析并返回一个 Say 语句节点。
  • 如果标记类型为 LISTEN，则解析并返回一个 Listen 语句节点。
  • 如果标记类型为 LOOKUP，则解析并返回一个 Lookup 语句节点。
  • 如果标记类型为 EXIT，则返回一个 Exit 语句节点。
  • 否则，假定它是一个跳转语句，并返回一个 Jump 语句节点。
返回
ASTNodePtr 指向解析后的语句节点的智能指针。
