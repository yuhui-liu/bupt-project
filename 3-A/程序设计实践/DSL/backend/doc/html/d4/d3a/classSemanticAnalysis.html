详细描述

语义分析类。分析程序的语义。

检查是否有重复定义的常量和过程；检查main过程是否存在。

参数
program指向 Program 对象的智能指针。
异常

将常量存储在 std::map 中，键为常量名，值为常量值。 将过程存储在 std::map 中，键为过程名，值为过程在程序中的索引。
