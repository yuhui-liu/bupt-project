解析并返回一个语句节点。 

该函数根据当前的标记类型解析不同类型的语句节点。
\begin{DoxyItemize}
\item 如果标记类型为 SAY，则解析并返回一个 \doxylink{classSay}{Say} 语句节点。
\item 如果标记类型为 LISTEN，则解析并返回一个 \doxylink{classListen}{Listen} 语句节点。
\item 如果标记类型为 LOOKUP，则解析并返回一个 \doxylink{classLookup}{Lookup} 语句节点。
\item 如果标记类型为 EXIT，则返回一个 \doxylink{classExit}{Exit} 语句节点。
\item 否则，假定它是一个跳转语句，并返回一个 \doxylink{classJump}{Jump} 语句节点。
\end{DoxyItemize}

ASTNode\+Ptr 指向解析后的语句节点的智能指针。 
\end{DoxyReturn}
